## Qu'est-ce qu'une norme ?

Une norme est une fiche technique qui décrit **comment construire un élément précis** dans Minecraft pour le projet Build The Earth France. Elle indique les blocs à utiliser, les dimensions, les proportions et éventuellement des variantes selon le contexte.

> [!INFO] Ce document est une introduction
> Si tu cherches comment soumettre une norme, consulte N1.2.

## Pourquoi suivre les normes ?

Les normes ne sont pas des obligations. Ce sont des recommandations validées par la communauté. Les suivre permet :

- D'avoir une cohérence visuelle entre les constructions de différents builders
- De gagner du temps — pas besoin de tout réinventer
- De faciliter le travail en équipe sur une même ville

## Comment lire une norme ?

Chaque norme suit le même format :

| Champ | Description |
|-------|-------------|
| ID | Identifiant unique (ex: N2.1) |
| Titre | Nom de l'élément normalisé |
| Statut | En cours / Validée / Dépréciée |
| Auteurs | Membres ayant rédigé la norme |
| Date | Dernière mise à jour |

## Statuts possibles

- **Validée** — Approuvée par le comité, recommandée pour utilisation
- **En cours** — En discussion, susceptible de changer
- **Dépréciée** — Remplacée par une norme plus récente, ne plus utiliser

> [!WARNING] Norme dépréciée
> Si une norme est marquée "Dépréciée", consulte la norme de remplacement indiquée dans la fiche.

## Niveaux de norme

Les normes sont organisées de façon hiérarchique :

- **N1** — Catégorie principale
- **N1.2** — Sous-catégorie
- **N1.2.3** — Norme spécifique
