## Qu'est-ce qu'une norme ?

Une norme est une fiche technique qui décrit comment construire un élément précis dans Minecraft pour le projet Build The Earth France. Elle indique les blocs à utiliser, les dimensions, les proportions et les variantes selon le contexte.

## Pourquoi suivre les normes ?

Les normes ne sont pas des obligations. Ce sont des recommandations validées par la communauté. Les suivre permet :

- D'avoir une cohérence visuelle entre les constructions de différents builders
- De gagner du temps sans avoir à réinventer chaque élément
- De faciliter le travail en équipe sur une même ville ou région

## Structure d'une norme

Chaque norme suit le même format.

| Champ | Description |
|-------|-------------|
| ID | Identifiant unique (ex : N2.1) |
| Titre | Nom de l'élément normalisé |
| Statut | Validée, En cours ou Dépréciée |
| Auteurs | Membres ayant rédigé la norme |
| Date | Dernière mise à jour |

## Statuts possibles

- **Validée** — Approuvée par le comité, recommandée pour utilisation immédiate
- **En cours** — En discussion, susceptible d'évoluer
- **Dépréciée** — Remplacée par une norme plus récente, ne plus utiliser

> [!WARNING] Norme dépréciée
> Si une norme est marquée Dépréciée, une norme de remplacement est indiquée dans la fiche. Utilise toujours la version la plus récente.

## Hiérarchie des normes

Les normes sont organisées en trois niveaux :

- **N1** — Catégorie principale (ex : Infrastructure routière)
- **N1.2** — Sous-catégorie (ex : Trottoirs)
- **N1.2.3** — Norme spécifique (ex : Trottoir standard 2 blocs)
